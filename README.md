<h2>BlogNock</h2>

<p>一个固执己见的中文技术博客的 <a href="https://github.com/search?q=Tampermonkey&type=repositories">Tampermonkey</a> 脚本集，致力于提升文章审阅的信息可见性</p>

### 站点支持

目前已支持以下博客站点

- [x] CSDN
- [x] 知乎
- [x] 简书
- [x] 思否（Segmentfault）
- [x] 博客园
- [x] 掘金
- [x] 51CTO
- [x] 腾讯云开发者社区
- [x] 阿里云开发者社区
- [x] 开源中国社区（OSCHINA）
- [x] BiliBili 专栏
