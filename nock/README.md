## CSDNock

### 功能

- 替换文章标识图片（原创/转载/- 翻译）
- 文章显示时间优化
- 文章阅读时长
- 移除底部推荐的 CSDN 下载
- 自动转载原链重定向

### 安装

[Greasy Fork 安装](https://greasyfork.org/zh-CN/scripts/493011-csdnock)

## ZhihuNock

### 功能

- 文章显示时间优化
- 文章阅读时长
- 知乎文章按钮位置调整
- 登录提示自动隐藏

[Greasy Fork 安装](https://greasyfork.org/zh-CN/scripts/493979-zhihunock)
